# Request Builder Module

**Location:** Main content area only

This module provides the main request builder interface for creating and editing HTTP requests.

## Structure

- `index.tsx` - Main RequestBuilder component (entry point)
- `context/` - RequestBuilder context provider
- `components/` - Sub-components (UrlBar, RequestTabs, ResponseViewer, etc.)
- `shared/` - Shared components used within request builder
- `types.ts` - Type definitions

## Usage

```tsx
import RequestBuilder from "@/modules/request-builder";
```
